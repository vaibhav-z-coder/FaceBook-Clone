## 📌 Overview

This **Facebook Clone** is a full-stack social media web application that replicates the core features of Facebook.
It demonstrates modern web development practices including authentication, REST APIs, and responsive UI design.

---

## ✨ Features

* 🔐 Secure User Authentication (Login / Register)
* 📝 Create, Edit & Delete Posts
* ❤️ Like & Comment System
* 👤 User Profile Management
* 🖼️ Profile & Cover Image Upload
* 📰 Dynamic News Feed
* 📱 Fully Responsive Design

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/facebook-clone.git

# Navigate into project
cd facebook-clone

# Install dependencies
npm install

# Start the server
npm start
```
